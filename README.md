# GTech Academy Networking Journey

A practical networking learning journey showing how networks evolved from basic communication to enterprise and Internet routing.

## Learning Approach

This repository follows the GTech Method:

Problem → Limitation → Solution → Implementation → New Challenge

## Covered Stages

✓ Computer Networks
✓ Static IP Addressing
✓ DHCP
✓ Routing
✓ Static Routing
✓ RIP
✓ VLANs
✓ Inter-VLAN Routing
✓ DHCP Relay
✓ OSPF
✓ BGP

## Labs

Every lab contains:

- Packet Tracer topology
- Screenshots
- Explanation
- Lessons learnt
