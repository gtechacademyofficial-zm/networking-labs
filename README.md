# GTech Academy Networking Labs

Welcome to the official GTech Academy Networking Laboratory repository.

This repository contains practical networking exercises designed to help students develop real-world networking skills through hands-on simulation, documentation, and problem-solving.

## About GTech Academy

GTech Academy is a technology education initiative focused on developing practical digital skills, building technology communities, and connecting learners with industry professionals.

## Learning Areas

The networking labs cover:

- Networking Fundamentals
- OSI and TCP/IP Models
- Ethernet and Switching Concepts
- IPv4 Addressing and Subnetting
- Routing Fundamentals
- VLAN Configuration
- Inter-VLAN Routing
- DHCP Services
- Network Security Fundamentals
- Enterprise Network Design

## Tools Used

- Cisco Packet Tracer
- Wireshark
- Linux Networking Tools
